# Portfolio Archive

> A premium scrollytelling portfolio built with **Next.js**, **TypeScript**, **Tailwind CSS**, **Framer Motion**, and **HTML5 Canvas**, featuring a cinematic scroll-driven experience for showcasing projects and professional case studies.

## 🚀 Live Demo

https://portfolio-archive-mak.vercel.app/


## ✨ Features

* Cinematic scrollytelling experience
* HTML5 Canvas image-sequence rendering
* Smooth scroll animations with Framer Motion
* Responsive and modern UI
* High-performance rendering pipeline
* Modular component architecture
* Optimized for modern browsers

## 🛠️ Tech Stack

* Next.js 14 (App Router)
* TypeScript
* Tailwind CSS
* Framer Motion
* HTML5 Canvas API

## 🎨 Design Philosophy

Premium editorial experiences and modern product storytelling inspire Portfolio Archive. Instead of relying on traditional page transitions, the interface uses scroll as the primary storytelling mechanism, combining cinematic motion, minimal design, and high-performance rendering to create an immersive user experience.

## ⚡ Getting Started

```bash
git clone https://github.com/Mariyamalikho/Portfolio-archive.git

cd portfolio-archive

npm install

npm run dev
```

Open **http://localhost:3000** in your browser.
